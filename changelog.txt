1.0.4
	If you first installed this game before March 7th, 2014, it is highly recommended that you reset your settings even if you've never changed them. You can do so by going to Settings -> Scroll Down -> Reset All Settings.
	You can now tap the hold box to hold your current Tetrimino.
		You are able to turn this feature off in the settings.
	Dragging up now cancels soft dropping. 
	Jetris now follows the Super Rotation System exactly!
	Jetris now implements Infinity exactly!
	Jetris now implements an exponential speed curve in Marathon Mode.
		You can choose between this system and the original (linear) system in the settings.
		As always, you are able to customize the curve.
	You can now choose which level to start at in Marathon Mode.
	Jetris now has the 3-corner T t-spin system.
		You can choose between this system and the original (Immobile) system in the settings.
	"T-spin" have been renamed to "Twist" to account for spins that are not with the T-Tetrimino in Immobile mode.
	Highscores now updates during the game.
		While displayed it doesn't fully save until the game is reset or paused.
	Restructured the game sequence to better display a loss.
	"Time's Up" message changed to "Finish".
	Improved performance for timer in time attack.
	Improved performance for displaying the grid.
	Improved performance when a Tetrimino is on delay.
	Improved performance for updating the locations of the active Tetrimino.
	Improved performance for loading the game.
	Default "Default Gravity" has been adjusted to 0.033. 
		You may want to change this setting manually (or press reset all settings).
	Reverted setting layout for integers.
	Fixed bug: Some settings are able to be changed such that it exceeds the cap.
	Fixed bug: Low timing resolution causing Tetriminos to fall erratically.
	Fixed bug: First block falling faster than normal on new game.
